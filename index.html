Projects


PIQA

Short for PyTorch Image Quality Assessment, PIQA is a Python package which implements a collection of measures and metrics for image quality assessment in various image processing tasks such as denoising, super-resolution, image interpolation, etc. It relies only on PyTorch and takes advantage of its efficiency and automatic differentiation.

ADOPPTRS

Acronym for Automatic Detection Of Photovoltaic Panels Through Remote Sensing, ADOPPTRS is a deep learning project whose goal is to detect photovoltaic panels in high-resolution satellite/aerial images.

VSOP Compiler

Implementation of a full source-code to machine-code compiler for the Very Simple Object-oriented Programming language inspired from COOL, the Classroom Object-Oriented Language.
